# 🌌 MARS Campus Intelligence Dashboard

An interactive full-stack application built for the **MARS Open Projects framework**. This system simulates an intelligent campus data orchestrator that ingests natural language student inputs, maps them semantically, and matches them to isolated campus subsystem endpoints (Library, Cafeteria, and Events calendars).

## 🚀 Key Architectural Features
* **Subsystem Matrix:** Isolated simulated endpoints modeling disjointed campus data nodes.
* **Semantic Middleware Brain:** An Express.js backend parsing logic groups.
* **Responsive Visual Dashboard:** Modern dark-mode web user interface.
